## 庸庸碌碌的每一天，靜下來讀一首詩

你有多久沒有讀詩了?每一天庸庸碌碌，打開瀏覽器不是要打開信箱看專案的進度報告不然就是要開始做給客戶看的簡報，每天忙得不可開交，是否好久沒有靜下心來泡一壺茶，細細品嘗一首詩呢?
